# Public Operating Strategy

This repository discloses the public operating principles of the OpenClaw trading workflow.

## Core Principles

- whitelist-based symbol universe
- bounded deployment size
- no leverage
- no short selling
- continuous monitoring with gated execution
- public reporting without exposing raw account data

## Public Disclosure Boundary

- publish holdings status and trade direction
- publish daily decision and review notes
- do not publish code, raw logs, balances, prices, quantities, or credentials
